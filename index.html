Introduction

  • Recent years have witnessed a greater flow of interest in effective computational methods, ranging from opinion mining, to subjectivity detection, to sentiment analysis.
  • These methods typically focus on the identification of public opinions, feelings, assessments, beliefs, sentiments, and conjectures in natural language.
  • Social media resources like Facebook and Twitter generate immense amounts of textual data on various topics.
  • This project aims to Louisiana based Twitter data to conduct user sentiment analysis.

Why Twitter Mining?

The Leviathan Twitter

  • More than 200m active users
  • New Tweets per second (TPS) record: 143,199 TPS
  • Average 5,700 TPS.
  • Typical day: more than 500 million Tweets sent
  • Short. 140-character message. Named as Tweets. User id is Twitter Handle.

Research Scopes

Objectives

The goal of this project is to develop a better algorithm for extracting location based real-time opinions on a particular set of interest group.

  • Specific Objectives
    • Review existing methods of real-time textual data mining algorithm
    • Develop algorithms for better text refining and knowledge distillation from social media data
    • Develop advanced interactive data visualization for real-time social media data
    • Develop user satisfaction matrix on particular interest group of countermeasures

require(RCurl)

accessURL <- "https://api.twitter.com/oauth/access_token"
authURL <- "https://api.twitter.com/oauth/authorize"
consumerKey <- "yourkey"
consumerSecret <- "yoursecret"
twitCred <- OAuthFactory$new(consumerKey=consumerKey,
                             consumerSecret=consumerSecret,
                             requestURL=requestURL,
                             accessURL=accessURL,
                             authURL=authURL)
setwd("yourfolder")
download.file(url="http://curl.haxx.se/ca/cacert.pem",destfile="cacert.pem")
